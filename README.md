🧠 Neuro Exam Monitor

Aplikasi simulasi ujian online berbasis HTML, CSS, dan JavaScript yang dilengkapi fitur monitoring aktivitas peserta seperti perpindahan tab (tab switch), deteksi AFK (Away From Keyboard), timer ujian, dan perhitungan skor otomatis.

🚀 Fitur
Login peserta ujian
Timer ujian
Navigasi soal (Next / Previous)
Perhitungan skor otomatis
Deteksi perpindahan tab browser
Deteksi pengguna tidak aktif (AFK)
Monitoring statistik aktivitas peserta
Halaman hasil ujian
📂 Struktur Project
project/
│
├── index.html
├── style.css
├── script.js
└── README.md
🛠 Teknologi
HTML5
CSS3
Vanilla JavaScript
▶ Cara Menjalankan
Opsi 1 - Langsung di Browser
Download project.
Pastikan file berikut berada dalam folder yang sama:
index.html
style.css
script.js
Buka file:
index.html
Aplikasi akan langsung berjalan di browser.
Opsi 2 - Menggunakan VS Code Live Server
Install extension Live Server di Visual Studio Code.
Buka folder project.
Klik kanan index.html.
Pilih:
Open with Live Server
Browser akan membuka aplikasi secara otomatis.
🌐 Cara Mengakses

Jika menggunakan Live Server:

http://127.0.0.1:5500

atau

http://localhost:5500

Jika menggunakan web server lain, sesuaikan dengan port yang digunakan.

📋 Cara Penggunaan
Masukkan nama dan NIM.
Klik Masuk Ujian.
Jawab soal yang tersedia.
Sistem akan:
Menghitung skor.
Mendeteksi perpindahan tab.
Mendeteksi aktivitas AFK.
Klik Submit atau tunggu timer habis.
Hasil ujian akan ditampilkan
